Update a user's budget

This endpoint allows a user to update a specific budget by providing the update type (add or subtract) and the amount to update. It also ensures validation of the current budget and the provided inputs.

Request Body schema: application/json
required

The budget update details.

UpdateType
required
string
Enum: "Add" "Subtract"

The type of update to perform (add or subtract).

BalanceType
required
string
Enum: "Checking" "Savings"

The type of balance to update.

UpdateSumInCents
required
integer

The amount to update, in cents.

Responses

Production server
